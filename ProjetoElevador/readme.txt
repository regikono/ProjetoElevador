Projeto de estudo para escolha de andaras de um elevador. 
Ao digitar entre 1 e 5 o projeto simular ir para o andar desejado. 
Se digitar andar inexistente, o projeto pedirá para digitar outro andar.
Não foi possível neste momento criar as funções utilziando IF, para poder verificar o andanr atual, mas poderá serm implemntado futurametne. 
Projeto criado em C#, apenas para estudos.
